### This is a simple HTML, CSS and JS project to make a fake credit card, allowing the user to change everything they want.

## Options to change:

+ Your Name
+ Bank Name
+ Card Numbers
+ Expiration Date
+ Gradient Card Color
+ Card Flag

2) Open the index.html in navigator
